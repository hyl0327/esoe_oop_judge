How to submit?

  1. Register an account at Bitbucket with your NTU Mail to get an academic plan.
  2. Create a new PRIVATE repository for this course.
  3. Share the repository to the account {{ bitbucket_account }} (only READ access permission is needed).
  4. Set up the Bitbucket settings in your profile.
  5. Clone the repository into your workspace (git clone).
  6. In the directory you just cloned, create a directory named {{ sample_problem.pk }} (mkdir {{ sample_problem.pk }}).
  7. Add the required submitted files (with your solutions, of course) specified in {{ sample_problem }} to {{ sample_problem.pk }}.
  8. Commit and push (git add, git commit, and then git push).
  9. Click the submit button in the problem detail page.
  10. See if there are any errors; if so, then go back to debug and submit again when you have done debugging.
  11. Do not forget to change {{ sample_problem.pk }} to the corresponding problem ID when you are solving other problems.
