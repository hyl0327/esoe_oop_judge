How to submit?

  1. Register an account at Bitbucket with your NTU Mail to get an academic plan.
  2. Create a new PRIVATE repository for this course.
  3. Share the repository (only reading permission is needed) to the account {{ bitbucket_account }}.
  4. Set up the Bitbucket settings in your profile.
  5. Clone the repository into whatever place you like (git clone).
  6. In the directory you just cloned, create a directory named {{ sample_problem.pk }}, which is the problem ID of {{ sample_problem.title }}.
  7. Add the required SUBMITTED files (with your solutions, of course) specified in {{ sample_problem.title }} into {{ sample_problem.pk }}.
  8. Commit and push (git add, git commit, and then git push).
  9. Click the submit button in {{ sample_problem.title }}.
  10. See if there are any errors; if so, then go back to debug and try it again when you are done.
Here, we use {{ sample_problem.title }} as sample. Do not forget to make corresponding changes when you are solving other problems.
